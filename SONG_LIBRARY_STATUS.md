## Strategy

### 1. Public Safe Repository
The public repository ships with **only public domain content**. This allows the project to be open-sourced without copyright infringement risks.

### 2. User Imports
Users can import their own songs using the import features. This places the responsibility of copyright compliance on the end-user for their personal library.
